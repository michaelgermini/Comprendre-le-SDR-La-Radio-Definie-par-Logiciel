# Chapitre 11 : Annexes

## 11.1 Glossaire complet

### Termes fondamentaux

#### A
- **ADC (Analog-to-Digital Converter)** : Convertisseur analogique-numérique. Transforme un signal analogique continu en signal numérique discret.
- **ADS-B (Automatic Dependent Surveillance-Broadcast)** : Système de surveillance automatique qui permet aux avions de diffuser leur position, altitude et identité.
- **AF (Audio Frequency)** : Fréquences audio, généralement 20 Hz à 20 kHz.
- **AGC (Automatic Gain Control)** : Contrôle automatique de gain. Ajuste automatiquement le niveau du signal.
- **AIS (Automatic Identification System)** : Système d'identification automatique pour les navires.

#### B
- **Bande passante** : Gamme de fréquences qu'un système peut traiter efficacement.
- **BER (Bit Error Rate)** : Taux d'erreur binaire. Probabilité qu'un bit soit reçu incorrectement.
- **BPF (Band Pass Filter)** : Filtre passe-bande. Laisse passer une bande de fréquences spécifique.
- **BPSK (Binary Phase Shift Keying)** : Modulation par déplacement de phase binaire.

#### C
- **Carrier** : Porteuse. Signal haute fréquence modulé pour transporter l'information.
- **CB (Citizens Band)** : Bande citoyenne. Bande radio publique autour de 27 MHz.
- **CW (Continuous Wave)** : Onde continue. Modulation utilisée en morse.
- **Cyclic prefix** : Préfixe cyclique. Extension ajoutée aux symboles OFDM pour combattre l'écho.

#### D
- **DAC (Digital-to-Analog Converter)** : Convertisseur numérique-analogique.
- **dB (Decibel)** : Unité logarithmique pour exprimer les rapports de puissance ou d'amplitude.
- **dBm** : Décibels par rapport à 1 milliwatt. Unité de puissance absolue.
- **Demodulation** : Processus inverse de la modulation. Extraction du signal original de la porteuse.
- **Doppler effect** : Effet Doppler. Changement de fréquence dû au mouvement relatif.
- **DSP (Digital Signal Processing)** : Traitement numérique du signal.
- **DVB-T (Digital Video Broadcasting - Terrestrial)** : Télévision numérique terrestre.

#### E
- **EIRP (Effective Isotropic Radiated Power)** : Puissance isotrope rayonnée effective.
- **ERP (Effective Radiated Power)** : Puissance rayonnée effective.
- **FFT (Fast Fourier Transform)** : Transformée de Fourier rapide. Algorithme efficace pour calculer la DFT.

#### F
- **FCC (Federal Communications Commission)** : Autorité de régulation américaine des communications.
- **FFT (Fast Fourier Transform)** : Transformée de Fourier rapide.
- **FIR (Finite Impulse Response)** : Réponse impulsionnelle finie. Type de filtre numérique.
- **FM (Frequency Modulation)** : Modulation de fréquence.
- **FPGA (Field Programmable Gate Array)** : Circuit programmable par l'utilisateur.
- **FSK (Frequency Shift Keying)** : Modulation par déplacement de fréquence.

#### G
- **Gain** : Amplification d'un signal, généralement exprimée en dB.
- **GMSK (Gaussian Minimum Shift Keying)** : Modulation GMSK utilisée dans GSM.
- **GNSS (Global Navigation Satellite System)** : Système global de navigation par satellite.
- **GPS (Global Positioning System)** : Système de positionnement global américain.
- **GSM (Global System for Mobile communications)** : Standard de téléphonie mobile 2G.

#### H
- **HackRF** : Plateforme SDR open-source pour émission et réception.
- **HF (High Frequency)** : Hautes fréquences, 3-30 MHz.
- **HPF (High Pass Filter)** : Filtre passe-haut.

#### I
- **I/Q (In-phase/Quadrature)** : Composantes en phase et en quadrature d'un signal.
- **IF (Intermediate Frequency)** : Fréquence intermédiaire.
- **IIR (Infinite Impulse Response)** : Réponse impulsionnelle infinie. Type de filtre numérique.
- **ISM (Industrial, Scientific and Medical)** : Bandes réservées aux applications industrielles.

#### J
- **Jamming** : Brouillage intentionnel d'un signal radio.

#### L
- **LNA (Low Noise Amplifier)** : Amplificateur basse bruit.
- **LO (Local Oscillator)** : Oscillateur local.
- **LPF (Low Pass Filter)** : Filtre passe-bas.

#### M
- **MIMO (Multiple Input Multiple Output)** : Entrées/sorties multiples.
- **Mixer** : Mélangeur. Combine deux signaux pour produire des fréquences somme et différence.
- **Modulation** : Processus de modification d'une porteuse selon un signal d'information.
- **Morse code** : Code utilisant des impulsions courtes (points) et longues (traits).

#### N
- **NF (Noise Figure)** : Figure de bruit. Mesure de la dégradation du rapport S/N.
- **NOAA** : Administration nationale américaine océanique et atmosphérique.
- **Nyquist theorem** : Théorème de Nyquist. Fréquence d'échantillonnage ≥ 2 × fréquence maximale.

#### O
- **OFDM (Orthogonal Frequency Division Multiplexing)** : Multiplexage par répartition orthogonale de fréquence.
- **OOK (On-Off Keying)** : Modulation tout-ou-rien.
- **OOK (On-Off Keying)** : Allumage/extinction de la porteuse.

#### P
- **PSK (Phase Shift Keying)** : Modulation par déplacement de phase.
- **PSD (Power Spectral Density)** : Densité spectrale de puissance.

#### Q
- **QAM (Quadrature Amplitude Modulation)** : Modulation d'amplitude en quadrature.
- **QPSK (Quadrature Phase Shift Keying)** : Modulation PSK quadratique.

#### R
- **RF (Radio Frequency)** : Fréquences radio.
- **RTL-SDR** : SDR basé sur le tuner RTL2832, populaire et abordable.
- **RSSI (Received Signal Strength Indicator)** : Indicateur de force du signal reçu.

#### S
- **Sampling rate** : Fréquence d'échantillonnage.
- **SDR (Software Defined Radio)** : Radio définie par logiciel.
- **SFDR (Spurious Free Dynamic Range)** : Plage dynamique sans produits parasites.
- **S/N (Signal-to-Noise ratio)** : Rapport signal/bruit.
- **SNR (Signal-to-Noise Ratio)** : Rapport signal sur bruit.
- **Spectrum analyzer** : Analyseur de spectre.
- **SSB (Single Side Band)** : Bande latérale unique.

#### T
- **TDMA (Time Division Multiple Access)** : Accès multiple par répartition dans le temps.
- **THD (Total Harmonic Distortion)** : Distorsion harmonique totale.

#### U
- **UHF (Ultra High Frequency)** : Ultra hautes fréquences, 300 MHz - 3 GHz.
- **USRP (Universal Software Radio Peripheral)** : Plateforme SDR professionnelle d'Ettus Research.

#### V
- **VHF (Very High Frequency)** : Très hautes fréquences, 30-300 MHz.
- **VSWR (Voltage Standing Wave Ratio)** : Rapport d'onde stationnaire.

#### W
- **Waterfall** : Représentation temporelle du spectre.
- **WFM (Wide FM)** : FM large bande pour la radio broadcast.

## 11.4 Flows GNU Radio prêts à l'emploi

### Récepteur FM stéréo

#### Description
Flowgraph pour recevoir la radio FM avec démodulation stéréo.

#### Blocs principaux
```
Source: Osmocom Source (RTL-SDR)
→ Low Pass Filter (bandwidth: 200kHz)
→ FM Demodulator
→ Stereo Demodulator
→ Audio Sink
```

#### Configuration
```python
# Paramètres
samp_rate = 2e6
freq = 93.5e6  # Station FM locale
audio_rate = 48e3

# Blocs
source = osmosdr.source(args="numchan=1")
source.set_sample_rate(samp_rate)
source.set_center_freq(freq)

lpf = filter.fir_filter_ccf(1, firdes.low_pass(1, samp_rate, 100e3, 10e3))

fm_demod = analog.quadrature_demod_cf(audio_rate/samp_rate)

stereo_demod = analog.stereo_demod()

audio_sink = audio.sink(audio_rate, "", True)
```

### Analyseur de spectre

#### Description
Visualisation temps réel du spectre avec waterfall.

#### Blocs principaux
```
Source: Osmocom Source
→ Stream to Vector
→ FFT
→ Complex to Mag^2
→ QT GUI Waterfall Sink
→ QT GUI Frequency Sink
```

#### Fonctionnalités avancées
- Ajustement automatique du gain
- Mesure de puissance par bin
- Export de données

### Émetteur ASK simple

#### Description
Émission de données binaires en modulation ASK.

#### Blocs principaux
```
Vector Source (data)
→ Repeat (symbol repetition)
→ Constellation Modulator (OOK)
→ Osmocom Sink
```

#### Paramètres
```python
# Configuration ASK
modulation = digital.constellation_ook()
modulator = digital.generic_mod(
    constellation=modulation,
    differential=False,
    samples_per_symbol=10,
    pre_diff_code=True,
    excess_bw=0.35
)
```

### Décodeur ADS-B

#### Description
Réception et décodage des signaux ADS-B.

#### Blocs principaux
```
Source: Osmocom Source (1090 MHz)
→ Low Pass Filter
→ Frequency Xlating FIR Filter
→ ADS-B Framer
→ ADS-B Decoder
→ Message Sink
```

#### Sortie
- Position GPS des avions
- Altitude, vitesse, cap
- Code ICAO

### Générateur de jamming (éducatif uniquement)

#### ⚠️ Usage strictement éducatif et légal uniquement

#### Description
Simulation de brouillage pour tests de résistance.

#### Blocs principaux
```
Noise Source
→ Throttle (rate control)
→ Osmocom Sink (low power)
```

#### Sécurité intégrée
- Puissance limitée
- Durée contrôlée
- Fréquence spécifique

### Récepteur satellite NOAA

#### Description
Réception d'images météo depuis satellites NOAA.

#### Blocs principaux
```
Source: Osmocom Source
→ Frequency Xlating FIR Filter (137 MHz)
→ FM Demodulator
→ Low Pass Filter (34 kHz)
→ File Sink (WAV recording)
```

#### Post-traitement
- Utilisation de WxToImg pour décodage APT
- Correction képlerienne
- Géolocalisation

### Simulateur de canal

#### Description
Simulation de conditions de propagation réalistes.

#### Blocs principaux
```
Signal Source
→ Channel Model (fading, noise)
→ Timing Offset
→ Frequency Offset
→ Receiver Chain
```

#### Modèles disponibles
- AWGN (bruit blanc gaussien)
- Rayleigh fading
- Rician fading
- Doppler shift

### Démodulateur QPSK

#### Description
Démodulation de signaux QPSK avec correction d'erreur.

#### Blocs principaux
```
Source
→ AGC
→ Costas Loop (carrier recovery)
→ FLL Band Edge (timing recovery)
→ QPSK Demodulator
→ Viterbi Decoder
→ BER Calculator
```

#### Performances
- Synchronisation automatique
- Correction d'erreur FEC
- Mesure de qualité
